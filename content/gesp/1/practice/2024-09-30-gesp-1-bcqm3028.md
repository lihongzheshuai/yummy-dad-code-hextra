## BCQM3028 输入-计算-浮点型格式化输出

### 题目要求

#### 描述

>给出圆的半径，求圆的直径、周长和面积。如果圆的半径是 $r$，那么圆的直径、周长、面积分别是$2 \times r$、$2 \times \pi \times r$、$\pi \times r^2$，其中约定$\pi = 3.14159$。

#### 输入

>输入包含一个实数$r$($0 < r \le 10,000$)，表示圆的半径。

#### 输出

>输出一行，包含三个数，分别表示圆的直径、周长、面积，数与数之间以一个空格分开，每个数保留 4 位小数。

#### 输入样例

>3.0

#### 输出样例

```console
6.0000 18.8495 28.2743
```

---

### 题目分析

简单的计算，练习点还是在于printf格式化输出的题，知识点在 ***%m.nf*** 的含义理解，已在[BCQM3027](https://www.coderli.com/gesp-1-bcqm3027/)中详细介绍。

---

### 示例代码

```cpp
#include <cstdio>
int main() {
    double a;
    scanf("%lf", &a);  // 读取圆的半径
    printf("%.4f %.4f %.4f", 2 * a, 2 * 3.14159 * a, 3.14159 * a * a);  // 输出直径、周长和面积，保留四位小数
    return 0;
}
```
